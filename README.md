# MyCMakeTestProject

## Makefile

- 🎈 use \<Alt> <Numpad: 0 0 9> for making tab (vsCode tab makes 4 spaces)

### Creating Makefile liux

```bash
touch Makefile
```

### Creating Makefile Windows

```sh
type nul > Makefile
#or
echo.> Makefile
```

### 🎆 MakeFile for linux

```make
prepare:
	rm -rf build
	mkdir build
```

### 🎇 MakeFile for Windows

```make
prepare:
	rmdir /s /q build
	mkdir build
```
